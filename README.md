## Getting Started

Update docker-compose.yml/.env file with OPEN AI API Key.

Run the Frontend + API server and run Mongo DB Docker containers -


```bash
docker compose up -d
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
